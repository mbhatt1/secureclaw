# `secureclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
secureclaw pairing list whatsapp
secureclaw pairing approve whatsapp <code> --notify
```

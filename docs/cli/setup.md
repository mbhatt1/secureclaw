# `secureclaw setup`

Initialize `~/.secureclaw/secureclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
secureclaw setup
secureclaw setup --workspace ~/.secureclaw/workspace
```

To run the wizard via setup:

```bash
secureclaw setup --wizard
```
